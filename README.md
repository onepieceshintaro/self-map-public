# 🗺️ 自分マップ（self-map-public）

**Phase 4「再選択（自己理解）」**フェーズ向けの自己理解ツール群。

## このアプリは何か

「自分はこういう人」を整理する場所です。

mood-tracker（Phase 1, 3）や 思考の整理ノート（Phase 1）で日々の記録を残し、
このアプリで**「自分の輪郭」を言語化**します。家族・職場・支援者に渡せる形
にも、自分の見返し材料にもなります。

## 含まれる機能（順次追加予定）

- ✅ 📋 **自分の取扱説明書**：調子のサイン・回復方法・強み・お願いなどを 1 枚に
- 🚧 🧭 **働き方の譲れない条件**：自分にとっての労働条件を整理
- 🚧 💪 **強みインベントリ**：過去の実例から強みを拾う
- 🚧 🎯 **価値観カードソート**：大事にする価値観を 5 つに絞る
- 🚧 🌧️ **再発のサインリスト**：自分のしんどさの早期警告を自分で書く

## 設計スタンス

- 書く・選ぶ系のアプリ（チャット系ではない）
- 全フィールド任意・空欄でも保存できる
- 「自分の言葉」が主、AI 補助は参考扱い
- 判定・診断・採点はしない
- 内容は本人のデータ。共有するかは本人が決める
